Training

DBS research group organizes intensive (1 to 3 days) courses on-demand, mostly in Finland. We have personnel and materials for the courses below. The courses can be tailored and new topics proposed. Our courses are given in Finnish or in English.

PostgreSQL

PostgreSQL fundamentals, SQL-dialect, database reliability, configuration, back-up and recovery, and database distribution.

MongoDB

MongoDB fundamentals, MongoDB query language, database reliability, database design, and database distribution.

Oracle Database

Oracle Database fundamentals, SQL-dialect, database reliability, configuration, back-up and recovery, and database distribution.

Cassandra

Cassandra fundamentals, CQL, database reliability, configuration, database design, and database distribution.

Database optimization

Physical and logical database design, query optimization, and knob tuning.

SQL basics

Querying and modifying data, data structures, basic expressions, table joins, ordering and grouping.

Advanced SQL

Analytics functions, window functions, common table expressions, and special data types.

PL/SQL

Functions, stored procedures, variables, and cursors.

Python basics

Fundamental concepts, object-oriented programming with Python, and database connectivity.

Linux basics

Kernel fundamentals, bash use and scripting, file system, user management, and process management.
